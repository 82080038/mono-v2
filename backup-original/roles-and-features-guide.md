## 📖 Panduan Lengkap Role & Features

**KSP Lam Gabe Jaya** memiliki sistem role-based access control yang komprehensif dengan 8 level akses yang dirancang untuk memenuhi kebutuhan operasional koperasi door-to-door.

## 👑 Hierarki Role System

### **Level 0: Creator (Pencipta Sistem)**
- **Username:** creator
- **Password:** creator123
- **Akses:** Full system access
- **Deskripsi:** Pencipta aplikasi dengan kontrol penuh

### **Level 1: Owner (Pemilik Bisnis)**
- **Username:** owner
- **Password:** owner123
- **Akses:** Kontrol bisnis keseluruhan
- **Deskripsi:** Pemilik bisnis dengan akses ke semua aspek bisnis

### **Level 2: Super Admin (Administrator Teknis)**
- **Username:** superadmin
- **Password:** superadmin123
- **Akses:** Manajemen sistem teknis
- **Deskripsi:** Administrator sistem dengan fokus teknis

### **Level 3: Admin (Administrator Sistem)**
- **Username:** admin
- **Password:** admin123
- **Akses:** Manajemen operasional
- **Deskripsi:** Administrator sistem operasional

### **Level 4: Manager (Manajer Operasional)**
- **Username:** manager
- **Password:** manager123
- **Akses:** Manajemen staff dan operasional
- **Deskripsi:** Manajer operasional dan staff

### **Level 5: Teller (Petugas Kasir)**
- **Username:** teller
- **Password:** teller123
- **Akses:** Layanan counter dan transaksi
- **Deskripsi:** Petugas kasir untuk layanan counter

### **Level 6: Staff (Karyawan Lapangan)**
- **Username:** staff
- **Password:** staff123
- **Akses:** Operasional lapangan
- **Deskripsi:** Karyawan lapangan untuk door-to-door service

### **Level 7: Member (Nasabah)**

- **Deskripsi:** Nasabah/koperasi member

## 🔄 **Role Progression**

### **Career Path Structure**
```
Member → Staff → Manager → Admin → Super Admin → Owner → Creator
```

### **Promotion Criteria**
- **Member → Staff:** 6 bulan aktif, training lengkap
- **Staff → Manager:** 1 tahun performa baik, target achieved
- **Manager → Admin:** 2 tahun experience, leadership skills
- **Admin → Super Admin:** Technical skills, system knowledge
- **Super Admin → Owner:** Business ownership, investment
- **Owner → Creator:** System development, technical expertise

- **Admin:** 8 hours, secure session

- **Teller:** 4 hours, normal session
